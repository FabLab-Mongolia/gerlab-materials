# GerLab Materials Library

Practical material-selection, compatibility, storage, and test-record guidance for GerLab.

> **Default rule:** Unknown or unlabeled material is **not approved** for processing.

This repository helps users choose and document materials. For equipment operation, use the [GerLab Machine Library](https://github.com/FabLab-Mongolia/gerlab-machines). For exposure controls, PPE, spills, and emergencies, use [GerLab Safety](https://github.com/FabLab-Mongolia/gerlab-safety).

## Material Status

- **Approved:** GerLab has identified the material and authorized a defined process.
- **Conditional:** Use requires specific machine, thickness, tooling, ventilation, or staff approval.
- **Test only:** Small supervised trial is required before normal use.
- **Prohibited:** Do not process by the stated method.
- **Unknown:** Quarantine until identified; never guess from appearance alone.

## Core Principles

1. Identify the exact material, grade, thickness, coating, adhesive, and supplier.
2. Check manufacturer information, SDS/TDS, machine limits, and GerLab approval.
3. Consider both processing safety and final-part performance.
4. Test small, record settings, and preserve traceability.
5. Stop if odor, flame, smoke, melting, tool load, finish, or machine behavior is unexpected.
6. Keep material records current as suppliers and formulations change.
